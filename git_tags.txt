Git tag | Static text pointer to a specific commit
Branches are dynamci pointers to the commit
---------------------------------------------------------------------------------------------------------------------
Staging is generally called as Release Branch
Production is generally called as Master Branch
 ---------------------------------------------------------------------------------------------------------------------
git tag | To see a list of all the tags
---------------------------------------------------------------------------------------------------------------------
git tag v1.0.0 | To create a new tag
---------------------------------------------------------------------------------------------------------------------
git lg | Show a list of all the logs
---------------------------------------------------------------------------------------------------------------------
Lightweight tags just has the name of the tag
Annotated tags   They have the name of the author, name of the tag, Date. They are also stored in the object folder
---------------------------------------------------------------------------------------------------------------------
git tags are not pushed with "git push command"
---------------------------------------------------------------------------------------------------------------------
git tag -a v1.0.0 -m "New tag"
---------------------------------------------------------------------------------------------------------------------
git push -v --tags | To push to remote along with the tags
---------------------------------------------------------------------------------------------------------------------
git push -v origin v1.0.1 | To push a specific version of the tag onto remote
