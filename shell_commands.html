/* This has all the Git commands */ Cd . will take to the current directory Cd.. will take to the parent directory Start . will open the current folder in explorer Ls -la will show a list of files along with the hidden files Cd --help will give a help on a given command Touch used to create a new file Echo used to print a value Echo "Hello world" > another-file.txt will write "Hello world" to the file Cat used to read the contents of a file Echo "Another world" >> another-file.txt will append "Another world" to the file Nano new-file.txt This will open the file in edit mode Ctrl+O + Enter Save the contents of the file Ctrl + x To exit Rm Remove file Rm -rf Remove folder and files within it
